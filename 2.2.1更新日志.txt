0620 / v2.1 更新日志
本轮主要改了 ShelfSheet 和 Library 两条链路。
ShelfSheet
右侧 Capture 贴纸云打开的 ShelfSheet 支持长按进入选择模式。
选择模式下贴纸会抖动，可多选。
已选贴纸显示绿色描边。
选择模式左上角显示“选择模式”。
右上角显示取消按钮和垃圾桶按钮。
删除前会弹窗确认。
ShelfSheet 支持在临时收集云和各云库之间切换。
标题左右增加箭头按钮。
支持左右滑动切换云库。
右滑/左滑到末尾可进入“添加云库”。
添加云库会弹窗命名，确认后创建并切换到新库。
每次打开 ShelfSheet 默认停留在“临时收集云”。
非选择模式下点击贴纸进入原始预览。
原始预览使用虚化背景，贴纸居中显示，支持双指缩放。
原始预览改为右上角圆形 x 按钮退出。
Library
删除了 类型 / 颜色 切换。
删除了分类筛选和颜色条。
删除贴纸类型、颜色、主色、颜色分组等相关逻辑。
Library 现在只保留搜索和贴纸网格。
贴纸名称改为新规则：自定义名称优先显示。
有自定义名称的贴纸永远排在自动编号贴纸前。
无自定义名称的贴纸按进入当前库的顺序显示为 贴纸 1、贴纸 2、贴纸 3。
删除或移动后编号自动保持连续。
清空名称并保存后，贴纸重新进入自动编号队列，并排到自动编号队列最后。

贴纸详情页删除了分类编辑。
贴纸详情页保留自定义名称、风格、保存相册、导出、删除等功能。
其他
Capture、Playground、Shelf 中无自定义名称的贴纸会显示兜底名称 贴纸，避免空白。
新增贴纸默认不再带英文来源名，会进入自动编号队列。
已做文本检查和旧功能关键词扫描。
当前 Windows 环境没有 Xcode/Swift 编译器，因此未做真实 iOS 编译验证。

0620 / v2.2.1 更新日志
Capture 页面云朵标题改为蓝色。
贴纸库继承选择模式：长按进入多选、贴纸抖动、绿色描边、取消和删除确认。
云库新增皮肤属性，并合并到库设置里的“库属性”；皮肤会同步影响云库 UI 识别。
Library 删除类型、颜色、分类筛选相关功能；贴纸改为自定义命名优先，未命名贴纸按进入顺序自动编号。
ShelfSheet 支持云库左右切换、滑动切换、添加云库、原始大小预览、虚化背景和圆形 X 退出。
Playground 主页重构为玩法入口：玩法1“随心贴”、玩法2“融合”；最近项目以静态叠加画廊在入口页后展示。
Playground 玩法页重排：右上草稿箱和保存，大面积编辑区，下方删除/工具/固化，再下方贴纸库，底部 tab 位置保持一致。
玩法1/玩法2 贴纸库云库 UI 与 Library 一致：云形图标、名称、贴纸计数，无小卡片底。
玩法1/玩法2 贴纸排序改为名称递增，计数显示为 已显示/总数。
玩法2 裁切模式进入时默认关闭，并改为画布大小裁剪。
工具栏默认收起，新增网格/网点、轴对称、自动对齐/吸附开关。
草稿箱新增：保存项目、未保存关闭项目、历史项目缓存，上限 10 个。
